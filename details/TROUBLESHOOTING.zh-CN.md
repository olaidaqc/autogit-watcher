# 常见问题 / 排查

## 1) 没有自动推送？

先看日志：`C:\Users\11918\Desktop\claude\autogit\autogit.log`

常见原因：

- `gh` 没登录：运行 `gh auth status`
- 网络/权限问题：日志里会有 `remote create failed` 或 `push` 相关报错
- 你刚改完文件还在持续保存：AutoGit 有 20 秒防抖（停下来才会 commit/push）

## 2) 为什么不把 `projects` 当一个总 Git 仓库？

可以，但不推荐：

- 一个大仓库会把所有项目绑在一起（一次 push/权限/可见性都会互相影响）
- 每个项目独立仓库更清晰，也更接近你“一个项目=一个仓库”的直觉

## 3) 我看到“会在本机安装模块（可逆）”是什么意思？

意思是：会把某些 PowerShell 模块装到你本机（通常是当前用户范围），以后命令能直接用。

这不是“系统更新”，也不会影响 Windows 更新；只是多装了一个可卸载的组件。

我们这套 AutoGit 不依赖额外安装模块（只用系统自带能力 + `git` + `gh`）。
